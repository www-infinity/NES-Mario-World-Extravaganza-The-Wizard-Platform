## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions. Any push to the `main` branch will trigger a deployment.

To see your changes live:
1. Create a pull request from your feature branch
2. Merge the pull request to `main`
3. GitHub Actions will automatically deploy the changes
4. Site will be live at: https://www-infinity.github.io/NES-Mario-World-Extravaganza-The-Wizard-Platform/

## Local Development

Simply open `index.html` in a web browser, or use a local web server:

```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server)
npx http-server
```

Then visit `http://localhost:8000`
